CSS117 Grid

A lightweight (~ 1Ko minified), minimalist and highly customizable SASS grid.

Presentation

CSS117 is a very smart and flexible grid where you don’t need to wrap individual rows nor to tag last columns with classes. It is highly customizable through variables

In the following example, one row become two rows in the medium breakpoint:

large-4 medium-12
large-4 medium-6
large-4 medium-6

Installation

  1. Download via bower (bower install css117) or from the Github repo (master.zip)
  2. @import in your main .scss file the partials css117-grid.scss (located in src/layout/)
  3. From it, copy from its two maps ($css117-grid-settings and $css117-grid-breakpoints) in your project before the @import and remove the !default flag

NOTE :

  • The two wrappers, .grid and .cols, are mandatory (the secret sauce of the grid)
  • The classes medium- and small- are created automatically from the map $css117-grid-breakpoints
  • The widest breakpoint is always called large and the narrowest small (both are mandatory)
  • You can add any other breakpoint between, directly in the map $css117-grid-breakpoints

Customize variables

Below are the variables from the $css117-grid-settings SASS map :

SASS variables Default Type Description
grid-class-name grid string Customize the first wrapper class name
cols-class-name cols string Customize the second wrapper class name
col-class-name col string Customize columns class name
nb-of-columns 12 integer Define the number of columns
max-width 960px pixels Define the max-width of the grid
gutter-width 30px pixels Define the width of gutters
grid-margin-top 0 pixels Define the margin top of the grid
grid-margin-bottom 30px pixels Define the margin bottom of the grid
grid-padding 0 pixels Define the padding of the grid (can be this way : 30px 10px 0 5px)
col-margin-top 0 pixels Define the margin top of columns
col-margin-bottom 30px pixels Define the margin bottom of columns
col-padding 20px pixels Define the padding of columns
col-font-size 1rem rem Define the base font-size inside columns
centered false boolean Choose true to center the grid on the page
full-width-on-small false boolean Choose true to have a unique fullwidth on the small breakpoint
offset-class false boolean Choose true if you need offset classes
offset-class-name offset string Customize name of offset classes. Works if offset is set to true
visibility-class false boolean Choose true if you need visibility classes
dev false boolean If set to true, it adds a pink transparent background-color behind elements and blue sides border to each columns
debug false boolean If set to true, it send debugging information to the ‘Terminal’

Fullwidth on small breakpoints

By default on the small breakpoint, each column of has different size, but in actual use, you probably want to set a unique fullwidth.

Then you don’t need the small class on your markup :

large-4 medium-6
large-4 medium-6
large-4 medium-12

Offset classes

Offset classes let you indent some columns.

large-12
large-4
large-offset-2
medium-no-offset
medium-6
large-6 medium-6
large-12
...
  • It is written BREAKPOINT-offset-NUMBER. Example : large-offset-2, medium-offset-7, etc.
  • It comes also no-offset classes : medium-no-offset, small-no-offset, etc.

Visibility classes

Visibility classes let you show or hide columns based on screen size. It creates a -hidden and -visible classes : large-hidden, medium-visible, etc.

large-12 medium-hidden
large-hidden medium-visible medium-6
large-12 medium-6
...
