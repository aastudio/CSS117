Media Queries mixin

A very flexible media queries mixin.

Presentation

  • Set as much as breakpoints you want
  • Choose to work mobile first or not
  • Set queries from either variables names, px, em or rem. Every query is convert into rem.

Installation

It’s all explained on the Get Started page.

small, medium, large, …

If you want to work with string names instead of numbers :

  • Copy the css117-mixin-mediaQueries.scss variables in your project and remove the !default flag :
  • or create you own map of breakpoints named $breakpoints (name is mandatory) :

NOTE : The widest breakpoint must be named large and the narrowest small. You can add any other breakpoint between.

Mobile First

If mobile-first is false

  • don’t declare the large breakpoint in your map
  • order your breakpoint from wide to narrow

If mobile-first is true :

  • don’t declare the small breakpoint in your map
  • order your breakpoint from narrow to wide

NOTE: The mixin automatically choose between max-width and min-width
