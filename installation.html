How to install a component

Installation

Via bower :

or by downloading the latest archive from the Github repo.

How to use

  • Components are located in the folder : src/partials/
  • @import the component in your main SASS file (usually main.scss)
  • Copy its variables (or just the one you need) in your project and remove the !default flag

For every components, each variables are gathered in a SASS map :

Example

To install the grid :

  • Copy and import in your project : src/partials/components/grid.scss
  • Copy its variables in your project and remove the !default flag

That’s all. Next step, see how to set up your html to use the grid.
